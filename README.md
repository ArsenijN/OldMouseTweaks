# Old Mouse Tweaks - `V0.0.1`

Bring the wanted QoL features for the Mouse Tweaks (link) mod!

(acts as backport on older version(s) of Minecraft for some of the features from More Mouse Tweaks (link))

---
## Prerequisites

Currently, OMT built with JDK 21, that may lead to incompatibility with Java 17 (default for 1.20.1 Minecraft). Java 21 works fine with 1.20.1, despite that it's **not** officially supported and **not** recommended. Downgrade to JDK 17 is soon

---

## Implemented features

> Ctrl+LMB: move the matching-item stacks between inventories (chest <--> inventory, hotbar <--> inventory)
